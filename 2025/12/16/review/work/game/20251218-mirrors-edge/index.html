《镜之边缘》：风景美不胜收，让我屏住呼吸

《镜之边缘》是一款独特的第一人称跑酷游戏。你在纯白明亮的未来都市中奔跑、跳跃，躲避追捕。它没有复杂的剧情和战斗，核心体验就是流畅移动带来的刺激与自由感。其极具风格化的视觉设计和沉浸感，让它成为游戏史上令人难忘的作品。





风从耳边掠过，脚步在屋顶回响。你几乎没有时间思考，只能判断下一个落点在哪里。跳跃、翻滚、奔跑，继续向前，不愿回头。敌人的追逐，警笛的尖啸，气流的呼呼声，都让你持续分泌肾上腺素。这种持续奔跑、不断向前的紧张感，构成了《镜之边缘》最直接、也是最核心的体验。


简介

纯粹的湛蓝色天空下，是一片以白色为主调的未来都市建筑群——玻璃幕墙泛着冷冽的光泽，摩天楼的线条利落又尖锐。阳光在建筑间切出明暗分明的棱角。在大楼的暗面，主角正在高空中的通风管道上奔跑、滑铲、转过拐角。阳光照耀，几只鸽子飞散开来，羽翼带起的风还没散尽，她便纵身跃向对面的楼宇——冒险在这一瞬彻底拉开了序幕。

《镜之边缘》的开场，至今想起都让我印象深刻。十几年以前，我还是个小孩，那时候总觉得第一人称游戏的屏幕右下角就该伸出一杆枪——毕竟我最爱玩的是《使命召唤4》，接触的也多是第一人称射击游戏。《镜之边缘》不一样。它以第一人称跑酷作为核心，至今极其少见，并且在当时一众脏兮兮的黄绿色调游戏中别具一格，令大多数玩家都印象深刻。最近，我生病躺在床上好久了，闲得发慌，于是又重温了一遍《镜之边缘》初代。另外，不得不说，串流真是一个好东西，能让我在床上躺着用平板电脑和手柄玩游戏了。

《镜之边缘》是由 EA DICE 开发、艺电发行的第一人称动作冒险游戏，最早于 2008 年推出。这是一款把“跑酷”作为核心玩法来设计的第一人称游戏，这在 2008 年非常罕见。更何况 DICE 的看家作品是《战地》系列，这一切似乎都和《战地》格格不入。

玩家扮演的信使名叫 Faith。女主角的面容非常亚裔，但并不难看，而且在续作里 Faith 甚至变得很酷、很好看。在当时那个年代的游戏社群中，“政治正确”这个概念几乎不存在，也没有纠结女主角的装束是不是为了迎合特殊群体。至少我挺喜欢这种干练的造型。

《镜之边缘》的故事发生在一座表面秩序井然、生活舒适的现代城市。表面上，这座城市犯罪率极低；与此同时，政府对通信、媒体、公共生活实施着全面监控。在这样的环境下，出现了一群被称为“信使”的人，他们用最原始的方式——奔跑，来躲避电子监控，传递实体信息。

主角绯丝从小生活在反政府家庭中，母亲在一次镇压示威的行动中丧生，这段经历让她对这座城市的体制非常反感。序章中，绯丝接到一条来自姐姐凯特的紧急求救信息。绯丝赶到现场后发现，一名市长候选人波普已经被谋杀，凯特作为一名警察被卷入案件，成了主要嫌疑人。为了洗清姐姐的嫌疑，绯丝开始在城市中奔走，寻找真正的凶手和背后的真相。随着调查深入，绯丝发现城市中正在秘密推行一个名为“伊卡洛斯计划”的行动。这个计划的目标，是彻底清除所有信使，切断不受政府控制的信息流。最终，克服重重阻碍，绯丝闯入城市的核心监控大楼，破坏了信息系统，救出了姐姐。

《镜之边缘》一代的故事并不复杂，流程也不长，而且是高度线性的，这一点我非常喜欢。线性结构让每一关都有精心编排的路线，而不是随意拼接出来的空间。你能明显感受到设计者在控制节奏：什么时候该全速冲刺，什么时候可以稍微放慢呼吸。相比之下，在一张空旷地图上随意圈出一块区域，再往里塞进一堆可有可无的收集品，更像是对关卡设计的一种亵渎。正是这种明确而克制的设计感，让我在游玩时产生了一种与设计者心有灵犀的交流感。这种体验非常难得，也让我印象深刻。

游戏里的反乌托邦设定并不复杂，世界观也没有被详细铺开，它更像是一个必要的背景，用来服务主要玩法和整体情绪基调，而不是试图构建一个庞大、严密的叙事体系。对于一个全新的 IP 来说，这样的取舍反而是合理的：先把核心体验立住，让玩家理解自己在为什么而奔跑，比急着讲清所有世界规则更重要。


美术

在《镜之边缘》中，美术不仅是装饰，它本身就是叙事和玩法的一部分。

我是处女座，正如刻板印象所说，我喜欢消毒水的味道，也喜欢整齐划一、干净利落的东西。这也是我一眼喜欢上《镜之边缘》美术设计的原因。城市的设计极为整洁，大量白色和灰色的建筑面板、干净利落的线条，让整个城市看起来像被“消毒”过。

游戏中，大量的白色空间带来了强烈的视觉统一感，同时也营造出疏离感，仿佛每一步都在光亮而无杂质的世界中前行。在此基础上，不同场景被赋予高饱和度的单色——洋红、明蓝、亮绿——既增加了视觉冲击力，也让城市的每一块区域更容易被记忆。每一栋建筑、每一个屋顶、每一个房间都像经过精心设计的画布。玩家奔跑其中时，不只是完成动作，更是在感受空间的布局与色彩的韵律。白色的秩序感与单色的强调相互呼应，给玩家带来了新奇的沉浸式体验。

我非常喜欢《镜之边缘》美术的另一个原因是，它的美丽与游戏主题高度契合，它的美术设计气质与游戏故事非常搭调。在《镜之边缘》中，美术设计在无声地讲述城市的故事，也与剧情紧密呼应。城市的极简与洁白，正反映了剧情中那个表面完美、实则高度管控的社会秩序。玩家在屋顶上奔跑时看到的整齐街区和无瑕建筑，时时刻刻都在提示玩家：这是一个被严格管理的世界，每一块砖、每一扇窗都在服从规则。每一次不走寻常路的跳跃、攀爬，都是玩家在用行动回应这个世界的限制。速度、节奏、手感，都是玩家与设计者之间的对话。这不正是第九艺术的精髓所在吗？是的，这是一种包含了美术在内的交互艺术。

奔跑在阳光照耀下的白色屋顶，映衬着远处高楼的剪影，你会不自觉地心跳加速，完全沉浸在这个气质独特而充满张力的世界中。每一处风景美不胜收，每一次奔跑都令人屏住呼吸。

不管这是有意为之，还是因为开发时间的限制，《镜之边缘》的城市里几乎空无一人。除了主角、敌人和少量 NPC，街道、屋顶甚至广场都显得格外寂静。正是这种孤独感，让整个城市呈现出一种遗世独立的清新氛围——这种感觉就像深秋的早晨，走出家门，吸入第一口清冽的空气，心里莫名涌起的一股沁人心脾的宁静感。这种环境让奔跑的每一步都显得格外鲜明，也让玩家在视觉和心理上更容易沉浸，感受到城市秩序与自由之间的微妙张力。

同时，因为游戏体量不大，游戏会通过一些细节来丰富世界观。正如很多人都知道的那样，游戏中的电梯是用来加载新区域的过渡场景。我很喜欢在等电梯的时候看一看其中的每日新闻。比如《每日城市》的这篇文章就挺有意思，标题是《对儿童的最新威胁：奔跑》——新闻中指出，衣物上的明显磨损、无法解释的擦伤、淤青甚至骨折，都是为非作歹的跑者身上的标志。另一篇文章则介绍了被杀的市长罗伯特·波普的基本背景，显然这位市长对这座城市的现状并不满意，这也成为了他被杀害的原因。


缺点

《镜之边缘》当然也并非没有缺点，甚至说，缺点不少。

游戏虽然有红色的视觉标识引导你前进，但到了后期，它的指引其实变得相当有限。比如有个场景让我印象深刻——在一个建造中的购物中心里，你需要从底层一路爬到高处。即便我已经通关了好几次《镜之边缘》，现在也还是想不起来那条路具体该怎么走。当时玩的时候，确实要花不少时间摸索攀爬路线。

此外，游戏的一些动作判定在当时也显得有点粗糙。毕竟是系列的第一次亮相，有时为了完成一个固定动作，我得反复尝试很多遍。第一人称视角下，想准确判断自己脚的位置其实并不容易。有时候你必须等到角色的脚踩到平台边缘，再按空格才能成功起跳——这种判定对操作要求有点高，甚至可以说有点“奇怪”。这也导致我在个别地方卡了很久。

好在这样的情况不算太多，整体流程还算流畅。不过偶尔出现一次，还是会让人有点头疼。

同时游戏中少有的战斗也显得有些笨拙和搞笑。动作僵硬，判定不清不楚。诸如此类的问题很多，玩家也都希望能看到一款更完善的续作。


续作

在 2016 年，经过8年《镜之边缘》迎来了它的续作《镜之边缘：催化剂》。

但说实话，我对这一作的印象并不是很好……坦诚讲，我确实玩过，但记忆中并没有通关。因为它犯了一个对我来说很致命的错误——它居然把开放世界元素融入了这款游戏。开放世界不乏优秀作品，但《催化剂》并不算特别优秀。那时候我是忠实的育碧玩家，玩过《看门狗》《刺客信条》《孤岛惊魂》……我对“开放世界”这个词都快 PTSD 了。我真的很不喜欢这套流水线做派，所以也没打算再重温它了。除非未来《镜之边缘》这个 IP 能重新复活，我可能才会再去玩一玩《催化剂》。不过我觉得，这个系列很可能已经被雪藏了……现在看来，从商业成绩上看，两部作品的表现都不算特别突出。而且估计 DICE 现在连《战地》都忙不过来了吧。

说到《战地》，《战地6》出了这么久，我一直没空玩，以后得找个机会体验一下。

说到这里，我突然想起 DICE 当年还有一个很有特色的系列——《战地：叛逆连队》。不过自从 2 代之后，3 代至今杳无音信。这个系列的制作人对这个 IP 感情很深，他甚至还在独自撰写游戏的后续剧情，并把它发到了网上。这位开发者表示，如今的游戏行业中，公司决策层为了确保经营安全、规避经济风险，往往会选择最理性、最依赖数据的商业决策。因此，即便这些老 IP 口碑很好、粉丝也不少，想让决策者开口重启它们，估计非常困难。

而在当年，游戏大厂还在尝试一些开创性的东西。至少在 2008 年，DICE 还在钻研他们的寒霜引擎。这个引擎如今已经成为了 DICE 的传家宝，而在当时，寒霜引擎在《叛逆连队》中的表现是开创性的。看看这爆炸，看看这坍塌的建筑。物理引擎的优秀其实在《镜之边缘》中也有体现——看看这破碎的塑料质感，看看这风声呼呼作响的感觉，实在太棒了。如今这种水平的物理效果也着实不多见。而我这次通关时，每次经过这种塑料帘，都要来回窜几次，感觉很棒。

DICE 当年在《镜之边缘》中打磨出的跑酷动作，也为后来具有突破性的《战地 :叛逆连队》、《战地 3》提供了基础。这套技术用来做战场破坏和士兵动作，效果刚刚好。


总而言之

第三次通关《镜之边缘》，只觉得现在的游戏画面越来越精细，成本越来越高，卖相越来越好，但能这样真正打动我的作品却不多。即便是现在，我依然很怀念十几年前——那个蓬勃向上、百花齐放的电子游戏黄金时代。
